// 7 sessions · python → ai

The full curriculum.

A clear path from your very first line of Python to training a model and shipping a chatbot. Each session builds on the last, with a challenge or project to make it stick.

01SESSION

Introduction to Python kickoff

  • What Python is and why it's everywhere
  • Cool, real examples of what code can do
  • Meet the REPL,where your code comes alive
02SESSION

Python Syntax & Basics challenge

  • Set up your Replit workspace
  • Variables, data types and clean syntax
  • Make Python do your math homework
  • End-of-class challenge
03SESSION

Data, Functions & Modules challenge

  • Strings, integers and lists
  • Write and call your own functions
  • Parameters and return values
  • Import modules like random & time
  • End-of-class challenge
04SESSION

Advanced Data & Control Flow quiz

  • More data structures and how to use them
  • Access, modify and loop through data
  • if / else logic and control flow
  • Tie it together into one real program
05SESSION

Intro to AI Concepts project

  • What artificial intelligence actually is
  • Real AI you use daily,feeds, self-driving, chatbots
  • How AI models learn from examples
  • First look at Teachable Machine
06SESSION

Train Your First AI project

  • What “training” really means
  • Build an image / gesture model step by step
  • Test it live with the class
  • Talk through AI bias,and why it matters
07SESSION

LLMs & Build Your Chatbot capstone

  • How large language models are trained
  • How chatbots actually work under the hood
  • Prompt-engineering, hands-on
  • Build & test your own bot with the Gemini API
